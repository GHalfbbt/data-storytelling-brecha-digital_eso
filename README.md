# 📊 Data Storytelling: Brecha Digital en la ESO - España

> *Transformando datos técnicos en historias que impactan y generan acción*


## 🎯 **Descripción del Proyecto**

Este proyecto forma parte de una **formación sobre Data Storytelling** y demuestra cómo transformar un análisis técnico de datos en una narrativa compelling que genere impacto y acción. 

Utilizando datos del **INE (Instituto Nacional de Estadística) - Dataset 43698** sobre competencias TIC en estudiantes de ESO, exploramos la brecha digital en España desde múltiples perspectivas: nacional, género, socioeconómica y territorial.

### 🎬 **La Propuesta de Valor**
- **Parte 1:** Análisis técnico profesional con Python
- **Parte 2:** Transformación en storytelling que "vende" la historia
- **Resultado:** Una masterclass práctica sobre la diferencia entre *hacer análisis* y *contar historias*

## 📊 **Hallazgos Principales**

### 🔍 **Datos Técnicos:**
- **43,698 estudiantes** de ESO analizados
- Solo **16.5%** alcanza nivel experto TIC
- **3 clusters territoriales** identificados mediante ML

### 🎭 **Historia que Contamos:**
1. **🚨 Crisis del 16.5%** - Solo 16 de cada 100 estudiantes dominan tecnología
2. **⚖️ Brecha de Género** - Las mujeres sufren 17.5 puntos de diferencia
3. **🗺️ Mapa de Desigualdad** - Tu código postal determina tu futuro digital
